About Shipment Predictor

Project Overview

The Shipment Predictor is an advanced machine learning application designed to predict the most appropriate supplier and account for shipments based on various characteristics like geographic locations, parcel dimensions, weight, and shipment category.

How It Works

Our application uses a hierarchical prediction model that analyzes shipment data through several steps:

  1. Data Preprocessing - Cleaning and formatting input data
  2. Geocoding - Converting addresses to geographic coordinates
  3. Category Classification - Categorizing shipments based on goods description
  4. Feature Engineering - Calculating distances and analyzing dimensions
  5. Hierarchical Prediction - First predicting the supplier, then the account
Technologies Used
  • Django web framework
  • Pandas for data manipulation
  • Scikit-learn for machine learning
  • Bootstrap for responsive design
  • DataTables for interactive results
Quick Start Guide
  1. Prepare your shipment data in CSV or Excel format
  2. Upload the file on the home page
  3. View the prediction results
  4. Download results as needed
